🛠️ Stack Tecnológica do Portfólio
Este portfólio foi construído do zero utilizando as melhores práticas do mercado para garantir máxima performance (SEO e tempo de carregamento) e uma UI/UX elegante:

    Framework: Next.js (App Router para otimização de rotas e Server Components)

    Linguagem: TypeScript (Tipagem estática para maior segurança)

    Estilização: Tailwind CSS (Design responsivo e utilitário)

    Componentes de UI: Shadcn/ui & Radix UI (Acessibilidade e design system)

⚙️ Como Executar o Portfólio Localmente

Caso queira explorar a estrutura de pastas, os componentes reutilizáveis ou o sistema de badges e alertas de manutenção que implementei, siga os passos abaixo:

1. Instale dependências:

```bash
git clone https://github.com/JudsonCiribelli/portfolio.git
```

2. Acesse a pasta do projeto:

```bash
cd portfolio
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```
